# 幹話產生器
自動，隨機，方便的幹話產生器，來玩玩看吧。

## Features
- 提供三種職業選擇
- 只要按兩下，1.選擇職業，2.按確認按鈕，立即送你一句幹話。
- 為方便您快速產出，點選圖片一樣ＯＫ，並貼心為您保留前次選項。
